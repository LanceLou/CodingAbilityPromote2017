# AutoCompleteInSearch(搜索框自动完成功能)

> 写代码用原生JS实现一个类似百度搜索框的自动完成控件，比如候选结果集arr=['aaa', 'abc', 'acc', 'dda',...]; 用户输入'a',下拉列表会出现以字母a开头的项'aaa', 'abc', 'acc'。
> 
> 要求：
> 
> 1. 鼠标或键盘可以选中候选结果到输入框> 2. 兼容IE8+/chrome> 3. 做题时间45分钟
